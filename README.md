## 命令行/Command Line
1. 登陆mysql控制台：
	* ```mysql -u [username] -p``` 
2. 创建一个数据库：
	* ```create database [database name];```
3. 使用新创建的数据库：
	* ```use [database name];```
4. 根据你的文件地址导入sql文件：
	* ```source [file location]/DDL/Init DB.sql;```
5.  检查是否成功：
	* ```show tables;```
